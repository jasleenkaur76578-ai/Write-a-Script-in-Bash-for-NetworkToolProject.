# 🌐 Network Information Tool (Bash)

A Bash scripting project developed on **Kali Linux** that collects and displays essential system and network information using standard Linux commands. The tool provides a quick overview of the system's network configuration, hardware, and resource usage.

## ✨ Features

- Display system hostname
- Show operating system information
- Display Linux kernel version
- Show local IP address
- Display MAC address
- Show default gateway
- Display configured DNS servers
- Check internet connectivity
- Show CPU information
- Display memory usage
- Show disk usage

## 🛠️ Technologies Used

- Bash Shell Scripting
- Kali Linux
- Linux Command-Line Utilities

## 📋 Commands Used

- `hostname`
- `uname`
- `hostname -I`
- `ip`
- `grep`
- `ping`
- `df`
- `free`
- `lscpu`

## 🚀 How to Run

```bash
chmod +x network_info.sh
./network_info.sh
```
